Kami berkomitmen untuk menghormati privasi data sesuai dengan UU No. 27 Tahun 2022 (UU PDP), UU ITE No. 11 Tahun 2008, dan standar internasional seperti GDPR.

🔍 2. Klasifikasi Data yang Dikumpulkan

A. Informasi Identifikasi Jaringan: Alamat IP Publik (IPv4/IPv6), Data Geolocation Kasar (Negara, Provinsi, Kota, Kode Pos), dan Metadata ISP (Provider, ASN, tipe koneksi).

🛡️ 1. Introduction and Legal Basis

SINTEL-HP is an Open Source Intelligence (OSINT) framework for defensive investigation, cybersecurity research, and digital forensic analysis. We align with GDPR, CCPA, and Law No. 27 of 2022 (UU PDP).

🔍 2. Data Classification and Collection

A. Network Identification: Public IP Address (IPv4/IPv6), Coarse Geolocation (Country, Region, City), and ISP Metadata (ASN, connection type).

B. Advanced Device Fingerprinting: Operating System details, Browser engine, CPU core count, estimated RAM, WebGL GPU Renderer, Battery telemetry, and Screen Orientation.

C. Precise Location (High-Accuracy GPS): Access to precise Latitude and Longitude occurs only with explicit manual permission from the target (Opt-in Consent).

💾 3. Data Retention and Security

Transient Data Flow: SINTEL-HP operates under a "Zero Persistence" policy. Captured data is streamed real-time via SSE and not stored on developer servers.

Security: Users must implement TLS/SSL (HTTPS) to prevent data interception during transmission.
